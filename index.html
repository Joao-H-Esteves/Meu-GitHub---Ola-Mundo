Curso de Linguagem C

C é uma linguagem que segue o padão ANCI, isto significa que toda liguangem C é padronizada,é considerada de nível médio, possui recursos tanto de baixo nível (linguagem da máquina) como de alto nível(linguagem humana). Permite a criação de bibliotecas, rotinas, manipulação de bits, bytes e endereços - os elementos mais básicos para a funcionalidade de um computador. C é uma linguagem compilada, fortemente tipada e pode ser compilada ou interpretada.
